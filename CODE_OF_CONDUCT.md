# Code of Conduct

## Our pledge

This project welcomes contributors and users of every background. We are
committed to a harassment-free, respectful experience for everyone taking part
in the project — in issues, pull requests, and any other project space.

## Our standards

We expect everyone to:

- Be respectful and considerate in language and conduct.
- Welcome differing viewpoints and accept constructive feedback gracefully.
- Focus on what is best for the project and its community.
- Show empathy toward other community members.

We do not tolerate harassment, personal attacks, demeaning or discriminatory
comments, or other unprofessional conduct.

## Standard adopted

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
full code of conduct. The summary above sets expectations; the linked document
is the complete, authoritative version, including the detailed standards and
enforcement guidelines.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Report unacceptable behavior to the project maintainer through GitHub — open an
issue if it is not sensitive, or use [private vulnerability reporting][report]
if it is. Reports are reviewed and handled confidentially. Maintainers may
remove, edit, or reject contributions, comments, or contributors that violate
this code of conduct.

[report]: https://github.com/jbcom/pdf-to-images/security/advisories/new
